# Instructions pour créer le screenshot

Créez un fichier image PNG de 588x438 pixels nommé "screenshot.png" 
dans ce dossier pour voir un aperçu du thème dans l'interface Drupal.

Le screenshot doit montrer :
- L'en-tête avec logo et navigation
- La zone principale de contenu
- Les couleurs et la typographie du thème

Vous pouvez temporairement ignorer cette étape - le thème fonctionnera sans screenshot.
